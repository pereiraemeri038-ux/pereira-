Cria uma aplicação de Galeria para Android completa, profissional e pronta para produção.

Tecnologias obrigatórias:
- Kotlin
- Jetpack Compose
- Material Design 3
- MVVM
- Hilt
- Room Database
- Coroutines
- StateFlow
- MediaStore API
- ExoPlayer
- Coil
- CameraX
- ML Kit
- BiometricPrompt

A aplicação deve ser compatível com Android 8.0 (API 26) ou superior.

### Interface
- Design moderno inspirado no Material Design 3.
- Interface elegante e rápida.
- Tema claro, escuro e automático.
- Cores dinâmicas.
- Animações suaves.
- Layout responsivo para telemóveis e tablets.
